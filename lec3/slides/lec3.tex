      \vspace{0.5cm}

      \textbf{Правильно}:

      Д просит П создать продукт

      П просит Х и М выполнить подзадачи для него

      М и Х возвращают результаты своих работ П

      П соединил результаты в продукт и вернул Д
    }

    \frame {
      \layersInRealLife
      \vspace{0.5cm}

    }

    }
